# Roblox-Minecraft-Tycoon-game-tech-tree-editor
Game goal (Minecraft-ish tycoon) Players buy packs in a shop using money. Packs give block cards → players place blocks in the world. Each placed block has a MoneyPerSecond (or equivalent) → adds to the player’s income loop every second. Player has an Inventory (cards not yet placed) and a Blocks folder (ObjectValues pointing to placed models). Money is tracked in leaderstats.Money. Shop/placement use RemoteEvents on the server. Tech Tree goal (what it controls) A visual system of upgrades/unlocks that guide progression: moneyBoost nodes: define a Block Name and a Boosted Income value (one-time or per level). custom nodes: define a RemoteEvent Name to trigger bespoke behavior. Costs can be money or blocks (later phase). Nodes have: Name, Description, Image ID(s), Type (Onetime or Repeatable with N levels), Functionality, Links (next/last) and Canvas position. Tech Tree Editor (Studio plugin, edit-mode) Docked GUI with: Canvas: add & drag nodes; draw links (later phase). Properties panel: edit fields; for Repeatable, pick level and edit that level’s data. Unsaved-change guard + Save button; prompt on switching node/level if there are pending edits (Save / Discard / Cancel). Export writes a ModuleScript (e.g., ReplicatedStorage.TechTreeData) containing: All nodes’ data (properties + per-level fields), Link relationships, Node positions (for reloading the layout). Runtime (game side) A TechManager reads TechTreeData: Applies moneyBoost effects (e.g., adjust per-block income or add multipliers). Fires custom RemoteEvents when such nodes are unlocked. Handles cost checks, unlock state, and progress saving. (Optional) Render the tree in-game for a player preview. Minimal data shape (example) TechTreeData = { nodes = { DirtBoost = { type = "Repeatable", levels = 3, functionality = "moneyBoost", upgrades = { [1] = { Name="Dirt I", Description="Start", Block="Dirt", BoostedIncome=2 }, [2] = { Name="Dirt II", Description="More", Block="Dirt", BoostedIncome=5 }, [3] = { Name="Dirt III",Description="Max", Block="Dirt", BoostedIncome=10 }, }, next = {"StoneUnlock"}, last = {}, pos = Vector2.new(120, 80), }, StoneUnlock = { type = "Onetime", functionality = "custom", RemoteEventName = "OnStoneUnlocked", cost = { money = 500 }, next = {}, last = {"DirtBoost"}, pos = Vector2.new(360, 80), }, } }
